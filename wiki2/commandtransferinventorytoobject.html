TransferInventoryToObject

Transfers a item [ARG 2] to a objects inventory [ARG 1] (And if set with the script running state [ARG 3])

Args
NameHint
itemUUID of item
objectuuid
runningtrue if you wish the transfered script to be running otherwise false
Return hints
  • ☑ Transfering running script
  • ☑ Transfering inventory
  • ☑ Invaild item uuid
  • ☑ Invaild object uuid
  • ☑ Unable to find inventory
  • ☑ Unable to find object
  • ☑ Invaild running
