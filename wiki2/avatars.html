Avatars

Avatars near me and Key2Name and Name2Key lookups

Get (4) / Set (2)Do (4)
GetProfileAbout

Requests the given avatars profile about me

GetProfileImage

Requests the given avatars profile image

Nearme

returns a list of all known avatars nearby

NearmeWithDetails

an improved version of near me with extra details
NearMeDetails is a object formated as follows

  • id
  • name
  • x
  • y
  • z
  • range

SetProfileAbout

Requests the bot updates its about me

SetProfileImage

Requests the bot updates its profile image

AcceptNext

Adds avatars to the one time accept list for interactions

IsOnAcceptNext

Checks if the avatar is on the accept next list (without using the token) for the given store

Key2Name

searchs the AV database if not found triggers a lookup

Name2Key

searchs the AV database if not found triggers a lookup
