SetEstateConfig

Sets the estate configuration for the current sim

Args
NameHintTypeExample valueSupported values
sun_hourthe hour of the day the sun should be at, 0 to 23Number12.0
is_sun_fixedtrue to fix the sun at the specified hour, false to have it move normallyBOOLfalse
is_externally_visibletrue to make the region visible on the map, false to hide itBOOLfalse
allow_direct_teleporttrue to allow direct teleports to the region, false to block themBOOLfalse
deny_anonymoustrue to deny access to anonymous users, false to allow themBOOLfalse
deny_age_unverifiedtrue to deny access to age unverified users, false to allow themBOOLfalse
block_botstrue to block known bots, false to allow themBOOLfalse
allow_voice_chattrue to allow voice chat, false to block itBOOLfalse
override_public_accesstrue to override public access settings, false to use parcel settingsBOOLfalse
override_environmenttrue to override environment settings, false to use parcel settingsBOOLfalse
Return hints
  • ☑ ok
  • ❌ Not an estate manager here
  • ❌ Unable to process sun hour value please use a number between 0 and 23
  • ❌ Sun hour value must be between 0 and 23
  • ❌ Unable to process is sun fixed value please use true or false
  • ❌ Unable to process is externally visible value please use true or false
  • ❌ Unable to process allow direct teleport value please use true or false
  • ❌ Unable to process deny anonymous value please use true or false
  • ❌ Unable to process deny age unverified value please use true or false
  • ❌ Unable to process block bots value please use true or false
  • ❌ Unable to process allow voice chat value please use true or false
  • ❌ Unable to process override public access value please use true or false
  • ❌ Unable to process override environment value please use true or false
