SetRegionFlags

Set the current region flags for the sim you are on

Args
NameHintTypeExample valueSupported values
setBlockTerraformtrue to block terraform, false to allow itBOOLfalse
setBlockFlytrue to block flying, false to allow itBOOLfalse
setAllowDamagetrue to allow damage, false to block itBOOLfalse
setAllowLandReselltrue to allow land reselling, false to block itBOOLfalse
setBlockPushingtrue to block pushing, false to allow itBOOLfalse
setAllowParcelJoinDividetrue to allow parcel join and divide, false to block itBOOLfalse
setAgentLimitthe maximum number of agents allowed in the simNumber100
setObjectBonusthe object bonus for the simNumber1.0
setMaturetrue to set the sim as mature, false to set it as generalBOOLfalse
setAdulttrue to set the sim as adult, false use the setMature settingBOOLfalse
Return hints
  • ☑ ok
  • ❌ Not an estate manager here
  • ❌ Unable to process block terraform value please use true or false
  • ❌ Unable to process block fly value please use true or false
  • ❌ Unable to process allow damage value please use true or false
  • ❌ Unable to process allow land resell value please use true or false
  • ❌ Unable to process block pushing value please use true or false
  • ❌ Unable to process allow parcel join divide value please use true or false
  • ❌ Unable to process agent limit value please use a number
  • ❌ Unable to process object bonus value please use a number
  • ❌ Unable to process mature value please use true or false
  • ❌ Unable to process adult value please use true or false
