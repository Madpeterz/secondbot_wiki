Inventory

Inventory controls for sending items and so on

Get (4) / Set (5)Do (18)
GetRealUUID

converts a inventory uuid to a realworld uuid
Needed for texture preview

InventoryContents

Requests the contents of a folder as an array of InventoryMapItem
Formated as follows
InventoryMapItem

  • id: UUID
  • name: String
  • typename: String

InventoryFolders

Requests the inventory folder layout as a json object InventoryMapFolder
Formated as follows
InventoryMapItem

  • id: UUID
  • name: String
  • subfolders: InventoryMapFolder[]

InventoryFoldersLimited

Requests folders limited to selected folder

CreateInventoryFolder

creates a folder in a folder

DeleteInventoryFolder

Attempts to Remove the given inventory folder

DeleteInventoryItem

Attempts to Remove the given inventory item

RenameInventory

renames a folder or inventory item

SetInventoryUpdate

Attachs an event for inventory changes

Attach

Attempts to attach the given inventory item

Detach

Attempts to Detach the given inventory item (or attachment point)

InventoryPurgeNotecards

Searchs the notecards folder for notecards, any older than 31 days are deleted.
Depending on the number of notecards this might require multiple calls!

MoveInventoryFolder

moves a folder to another folder

MoveInventoryItem

moves an item to another folder

Outfit

Replaces the current avatar outfit with the My Outfits/[NAME] folder

RezObject

rezs the item at the bots current location

RezObjectOnPos

rezs the item at the target location

RezObjectParcelCenter

rezs the item at the parcel center

SendFolder

Sends a folder to an avatar

SendFolderByPath

Sends a folder to an avatar

SendItem

sends a item to an avatar

SendItemByPath

sends a item to an avatar via a path

SendItemDirect

sends a item to an avatar with extra steps to bypass inventory searchs

SwapOutfit

Replaces the current avatar outfit with the contents of the given folder

SwapOutfitByPath

Replaces the current avatar outfit with the contents of the given folder

TransferInventoryToObject

Transfers a item [ARG 2] to a objects inventory [ARG 1] (And if set with the script running state [ARG 3])

UploadMediaWave

Uploads a new sound file to inventory
